# babelpish

## nlp

파트1 바벨파이에서 발표했던 자료가 있습니다.

- bag of words를 이용한 긍부정 분류
- lda를 이용한 문서클러스터링 & 뉴스 크롤링

## cs20si

파트2 바벨쿠키에서 공부중인 Stanford의 Tensorflow 강의를 쳐보고 있습니다.
